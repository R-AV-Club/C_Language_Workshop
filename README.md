# C Language Workshop

A beginner-friendly C programming workshop designed for students who are starting their programming journey.

This repository contains the complete notes, examples, and practice problems covered during the workshop. The material is written in simple language and focuses on building programming logic step by step.

The goal of these notes is not only to teach C syntax but also to help students develop problem-solving skills that will be useful in Data Structures and Algorithms, Competitive Programming, and Technical Interviews.

## How to Use These Notes

If you are completely new to programming:

1. Read the notes day by day.
2. Type every program yourself instead of copying.
3. Execute the programs and observe the output.
4. Complete the homework questions at the end of each day.
5. Try modifying programs and experimenting with different inputs.

Programming is a practical skill. Reading alone is not enough. The more programs you write, the better you become.

## Practice First, Memorize Later

Do not try to memorize programs.

Focus on understanding:

- Why a program works
- How variables store values
- How conditions make decisions
- How loops repeat tasks
- How functions organize code

Once you understand the logic, writing programs becomes much easier.
